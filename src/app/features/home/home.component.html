Welcome to ActorTrack!

The platform is designed to manage and track information related to actors, performances, and contracts in a production environment. It follows the modern software architecture and utilizes a range of powerful technologies to deliver a seamless user experience:
  • MongoDB: The platform uses the latest stable version of MongoDB, with Mongoose for database interaction. MongoDB Atlas is utilized for cloud database management, providing a scalable solution.
  • Express.js: The backend is built with the latest version of Express.js. It ensures smooth routing and efficient handling of HTTP requests. Authentication and authorization are implemented using JWT for secure access control.
  • Angular: The frontend is developed using Angular. Material UI is used to create a responsive and clean user interface.
  • Swagger: A RESTful API is developed using Express.js, and API documentation is created using Swagger for easy understanding and usage.
  • GitHub: The project is managed and stored on GitHub for version control.
